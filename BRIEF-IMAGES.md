# ТЗ дизайнеру: 12 фото для лендинга start.it-kiber.ru

## Общие правила (читать ОБЯЗАТЕЛЬНО)

1. **Формат:** JPEG, sRGB, прогрессивный. Сжатие — до 250 КБ каждый файл (на retina-фото — до 400 КБ). Без водяных знаков.
2. **Имена файлов — точно как ниже.** Каждый файл должен называться слово-в-слово, иначе сайт не подхватит. Прислать одной папкой.
3. **Способ создания на выбор:**
   - **AI-генерация** (Midjourney v6/v7, DALL-E 3, Sora image, Recraft, Flux) — промпт прилагается.
   - **Реальная съёмка** в наших филиалах — тогда нужны письменные согласия родителей по 152-ФЗ.
4. **Главное правило по детям:** лица детей **не должны быть узнаваемы**. Допустимые приёмы:
   - дети сняты со спины / полуоборота / в три четверти со склонённой головой;
   - крупный план — руки на клавиатуре, глаза на экран, лицо обрезано;
   - лёгкий motion blur лица;
   - для AI-фото — генерировать **без чёткой детализации лиц** (`face slightly blurred, soft focus on face`).
   Иначе мы выдаём чужих детей за «наших учеников» — это юридически и этически некрасиво.
5. **Брендовые цвета KIBERone** (желательно присутствие хотя бы одного на фото):
   - Синий — `#0052CC` (стены, толстовки, наклейки)
   - Жёлтый акцент — `#FFD93D` (одежда, элементы интерьера)
   - Тёмный — `#0F1419` (мониторы, столы)
6. **Стиль:** светлый, чистый, минималистичный. Естественный свет от окон. Современная техника (моноблоки, ноутбуки белые/серые). Никакой ретро-эстетики и «школьных парт».

| Параметр | Значение |
|---|---|
| Размер | **2400 × 900 px** |
| Соотношение | 24:9 (на мобильном автоматически кропнется до 16:9 по центру) |
